## Описание программы


**Movie Library** - это GUI-приложение для управления списком фильмов. Приложение позволяет добавлять фильмы, фильтровать их по жанру, сохранять и загружать данные в формате JSON, а также удалять отдельные записи или весь список целиком.


### Возможности:
- ✅ Добавление фильмов
- ✅ Фильтрация фильмов по жанру 
- ✅ Сохранение данных в JSON-файл
- ✅ Загрузка данных из JSON-файла
- ✅ Очистка всего списка 
- ✅ Валидация вводимых данных (проверка пустых полей, рейтинг, год)


## Инструкция по запуску
- python main.py

### Требования:
- Python 3.7 или выше
- Установленная библиотека Tkinter (обычно идёт в комплекте с Python)
- Установленная библиотека json


### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
   ```bash
   cd myProject
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```


## Примеры использования


### Добавление книги:
1. Заполните все поля формы:
2. Нажмите кнопку **"Добавить фильм"**
3. Фильм появится в списке внизу окна


### Сохранение и загрузка:
- Нажмите **"Сохранить в JSON"** - данные сохранятся в файл `movies.json`




## Валидация данных


Приложение проверяет:
- Все поля должны быть заполнены
